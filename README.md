# Moth-Flame Optimization (MFO) Project

This project contains a simplified implementation of the Moth-Flame Optimization algorithm using Python. It includes:

## Structure
- `src/` : Python modules (MFO algorithm and objective functions)
- `examples/` : Colab notebook demo
- `requirements.txt` : Required Python packages

## Demo
- Run the notebook `examples/mfo_demo.ipynb` in Google Colab to see the convergence curve.
- Run the notebook `examples/MFODetails.ipynb` in Google Colab to see the progress of optimization.
